# Passione-Agents

**Claude Code用の自律型マルチエージェントシステム**

「覚悟」とは、暗闇の荒野に、進むべき道を切り開くことだ。

Passione-Agentsは、複雑なプロジェクトを複数の専門エージェントが協力して遂行する、
Claude Code向けのマルチエージェントフレームワークです。

## 特徴

- **役割分担**: 設計、実装、レビュー、監視、調査を専門エージェントが担当
- **統一された進捗管理**: MISSION.mdでチーム全体の状況を共有
- **段階的なワークフロー**: 設計→実装→検証の明確なフェーズ
- **即座に使える**: セットアップスクリプトで簡単にインストール
- **実例豊富**: Kaggleコンペなど、実践的なサンプルプロジェクトを収録

## クイックスタート

### 1. インストール

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/passione-agents.git
cd passione-agents

# セットアップスクリプトを実行
bash setup/setup.sh
```

### 2. エージェントの起動

Claude Codeで以下のように起動します：

```
# ブチャラティ（Orchestrator）を起動
/bucciarati

# 個別のエージェントも起動可能
/fugo        # 設計
/giorno      # 実装
/abbacchio   # レビュー
/narancia    # 調査
/mista       # 監視
```

### 3. プロジェクトを始める

```markdown
# ボス（あなた）からの指示例
「Kaggleのタイタニックコンペで上位10%を目指すプロジェクトを作ってくれ」

# ブチャラティが受領
「了解した。まずフーゴに設計させる」

# フーゴが設計書（MISSION.md）を作成
# ジョルノが実装
# アバッキオがレビュー

# 最終報告
「ボス、任務完了だ」
```

## チームメンバー

| 名前 | 役割 | 専門分野 |
|------|------|---------|
| **ブチャラティ** | Orchestrator（統括） | プロジェクト管理、タスク分配 |
| **フーゴ** | Planner（設計） | アーキテクチャ設計、タスク分解 |
| **ジョルノ** | Developer（実装） | コーディング、ファイル作成 |
| **アバッキオ** | Reviewer（検証） | コードレビュー、テスト実行 |
| **ナランチャ** | Researcher（調査） | 技術調査、ドキュメント検索 |
| **ミスタ** | Monitor（監視） | 進捗監視、問題検出 |
